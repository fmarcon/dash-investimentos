# Dashboard de Investimentos

## Visão geral

Dashboard de Investimentos é uma aplicação financeira para investidores individuais acompanharem compras, vendas e a evolução do patrimônio em ativos de bolsa.

O usuário autenticado poderá cadastrar operações de compra e venda de ações e visualizar lucros, prejuízos e evolução patrimonial. O sistema também terá um perfil administrativo para ajustes e manutenção de registros de usuários e transações.

## Público-alvo

- Investidores individuais que desejam controlar e analisar seus investimentos em bolsa.
- Administradores do sistema responsáveis por suporte, revisão e ajustes de dados.

## Valor entregue

- Interface de login e cadastro de usuário.
- Tela inicial adaptada por perfil: investidor ou admin.
- Registro de operações de compra e venda.
- Dashboard financeiro com evolução de patrimônio e indicadores de performance.
- Admin simples para corrigir dados ou ajustar registros de usuários.

## Qualidades do produto

- Financeiro moderno: visual limpo, foco em dados e clareza.
- Segurança básica de autenticação e separação de papéis.
- Priorização de MVP funcional para acelerar a primeira entrega.

## Restrições

- MVP inicial deve focar em autenticação, registro de transações e dashboard por perfil.
- Deve suportar investidor individual e administrador, sem integração externa inicial.
- O projeto pode usar banco de dados local ou serviço simples, mas deve persistir usuário e operações.
