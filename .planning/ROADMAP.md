# Roadmap do Projeto

## Fase 1 — MVP de operações e dashboard

- Configurar autenticação de usuário e rotas protegidas.
- Implementar cadastro e login de investidor e admin.
- Criar modelo de transação de compra/venda.
- Construir dashboard do investidor com resumo de carteira.
- Criar tela administrativa básica.
